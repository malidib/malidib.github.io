لا يمكن مُعالجة البحث لعدم توافُر محتواه

المستند المُرسَل لا يتضمّن نصّ البحث أو ترجمته، بل يحتوي فقط على ملاحظة باللغة الإنكليزية تُفيد بعدم وجود معادلات LaTeX.

لإنجاز المطلوب بدقّة (مراجعة الترجمة فقرةً فقرةً، تصحيح اللُّغة والمصطلحات العلميّة، تحسين البُنية، والحفاظ على أي بنى LaTeX)، أحتاج إلى:

  • النصّ الكامل للبحث المُترجَم إلى العربيّة بصيغة HTML (أو النصّ العادي وسأتولّى أنا تغليفه ضمن HTML مُحسَّن).
  • اختياريًّا: النسخة الأصليّة بالإنكليزية للمراجعة الاصطلاحيّة الدقيقة.

إذا لم يكن لديك HTML جاهز، انسخ النصّ المُترجَم مباشرة هنا. سأقوم بتنظيفه، تصحيح الأخطاء النحويّة والأسلوبيّة والعلميّة، توحيد المصطلحات، إضافة التشكيل الضروريّ فقط، وتحسين العرض دون الإخلال بالمحتوى أو ببُنى LaTeX إن وُجدت.

مثال على كيفية الإرسال:

  • إرسال HTML كامل يبدأ بـ <html>...</html>.
  • أو إرسال النصّ المُترجَم كما هو، وسأُعيد تنسيقه ضمن هيكل HTML بسيط وأنيق.

حالما تُزوِّدني بالمحتوى، سأُعيد إليك النسخة المُصحَّحة ضمن هيكل HTML مُحسَّن، مع الحفاظ على أي بنى LaTeX سليمة وتصحيح ما يلزم منها إن ظهرت أخطاء واضحة.
